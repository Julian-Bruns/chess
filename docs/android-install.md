# Install Chessfish On Android

This is the shortest install path for people who just want to play chess.

## Download

Open this link on the Android phone:

https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-android.apk

If the browser asks what to do with the file, choose **Download**.

This link starts working after the GitHub workflow has published the first `latest` release. If it shows a GitHub 404 page, the release has not been built yet, the APK was not published, or the repository/release is private. Private GitHub release downloads are not visible to players who are not signed in with repository access.

## Install

1. Open the downloaded file named `chessfish-android.apk`.
2. If Android says the browser is not allowed to install apps, tap **Settings**.
3. Turn on **Allow from this source**.
4. Go back and tap **Install**.
5. Open **Chessfish** from the app drawer.

Android shows that warning for any app installed outside the Play Store. You only need to allow it for the browser or file manager you used to open the APK.

## Update Later

Open Chessfish and press the download button in the top bar. Android may ask for install confirmation again.

## If The Link Does Not Work

The direct download only works when the GitHub `latest` release asset is public. If this link shows a GitHub 404 page, make sure the repository is public and the Daily app builds workflow has published the `latest` release.
