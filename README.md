# Chessfish

Chessfish is a simple chess app for playing White against local Stockfish.

## Install

| Platform | App version | Download path |
| --- | --- | --- |
| macOS | `0.1.0` | Published after Developer ID signing and notarization succeeds |
| Windows | `0.1.0` | https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-windows.exe |
| iOS | `0.1.0` | https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-ios.ipa |
| Android | `0.1.0` | https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-android.apk |

### macOS

The macOS DMG is published only after Developer ID signing and notarization succeeds. Once available, download the latest signed and notarized DMG, open it, and drag Chessfish into Applications:

https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-macos.dmg

Release signing setup is documented in [docs/macos-signing.md](docs/macos-signing.md).

### Windows

Download the latest Windows installer and open it:

https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-windows.exe

### iOS

Download the latest signed iOS build:

https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-ios.ipa

### Android

Download the latest APK on your phone, open it, and tap Install:

https://github.com/Julian-Bruns/chess/releases/download/latest/chessfish-android.apk
